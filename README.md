# AutoIt

Repositório feito para guardar alguns scripts e programas úteis feitos em AutoIt e resolvem alguns problemas do dia a dia.

## Programas
- Ajudante de Pick para Dota 2
    - Faz um web scrap no site do DotaBuff, cria um banco de dados no computador e mostra quais os melhores personagens para pegar, dependendo do adversário.

## Contribuição

Você pode contribuir com esse repositório, basta fazer um fork, colocar seu script e fazer um Pull Request.

## Idéia

Alguns anos atrás eu jogava o jogo Dota 2 e queria um programa que mostrava quais os melhores heróis para pegar em uma partida. Foi ai que descobri essa linguagem e sua utilidade em fazer Web Scrapers.
